# AIPEX Accessibility Standards

Target:

WCAG 2.2 AA

Goal:

WCAG AAA wherever practical.

---

Requirements:

✓ Semantic HTML

✓ Keyboard navigation

✓ Visible focus states

✓ Skip links

✓ Alt text

✓ Proper heading hierarchy

✓ Accessible forms

✓ Color contrast compliance

✓ Reduced motion support

---

Minimum tap target:

44px

---

Accessibility is not optional.

It is part of product quality.
