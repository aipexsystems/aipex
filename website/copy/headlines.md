# Headline Standards

All headlines should be:

✓ Outcome-focused

✓ Human-centered

✓ Industry-aware

✓ Clear and concise

✓ Free of technical jargon

✓ Trust-oriented

Avoid:

✗ Buzzwords

✗ AI hype language

✗ Developer terminology

✗ Feature lists

✗ Aggressive marketing language

Rule:

Speak to business transformation—not technology implementation.
